================================================================
 FBNeoRageX — Steam Deck 컨트롤러 설정 가이드
================================================================

■ 기본 조작 (별도 설정 불필요)
──────────────────────────────────────────────────────────────
  D-패드       → 게임목록 위/아래 탐색
  A 버튼       → 선택한 게임 실행
  좌측 스틱    → 게임 플레이 방향 이동 (에뮬레이션 중)
  우측 스틱    → 마우스 커서 (UI에서 사용)

■ L4/R4/L5/R5 & 오른쪽 트랙패드 — Steam Input 설정
──────────────────────────────────────────────────────────────
  Steam Deck에서 아래 버튼을 설정하려면:

  1. Steam 버튼 → 컨트롤러 설정 → FBNeoRageX 레이아웃 편집

  2. 다음과 같이 매핑:
     ┌──────────────┬────────────────────────────────────────┐
     │ 스팀덱 버튼  │ 기능                                   │
     ├──────────────┼────────────────────────────────────────┤
     │ L4 (뒷면 L)  │ 키보드: Tab  → GUI 패널 전환           │
     │ R4 (뒷면 R)  │ 키보드: Esc  → 게임 종료/메뉴 복귀     │
     │ L5 (뒷면 L2) │ 마우스: 오른쪽 클릭 → 컨텍스트 메뉴    │
     │ R5 (뒷면 R2) │ 마우스: 왼쪽 클릭  → UI 요소 클릭      │
     │ 우측 트랙패드│ 마우스 이동 (터치패드 모드)             │
     └──────────────┴────────────────────────────────────────┘

  3. 설정 후 "적용 및 저장" → 커뮤니티 레이아웃으로 공유 가능

■ Steam Input 빠른 설정 방법
──────────────────────────────────────────────────────────────
  게이밍 모드에서:
    Steam 버튼 → 게임 진행 중에도 컨트롤러 아이콘 클릭
    → "Edit Layout" → 각 버튼 항목 클릭해서 변경

  데스크톱 모드에서:
    Steam 앱 → 라이브러리 → FBNeoRageX 우클릭
    → 속성 → 컨트롤러 → 커스텀 구성 편집

■ 키보드 단축키 (연결된 키보드 사용 시)
──────────────────────────────────────────────────────────────
  Tab       → 게임 일시정지 / GUI 전환
  Esc       → 게임 종료 → 목록으로
  F1        → 세이브스테이트 (슬롯 1)
  F5        → 로드스테이트 (슬롯 1)
  F11       → 패스트포워드 (배속)
  F          → 전체화면 전환

■ 넷플레이
──────────────────────────────────────────────────────────────
  OPTIONS → NETPLAY → HOST GAME 또는 CONNECT
  Steam Deck ↔ Windows 크로스플레이 지원
